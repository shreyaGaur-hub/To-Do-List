# To-Do List App 📝

A sleek, responsive, and functional To-Do List application built with HTML, CSS, and Vanilla JavaScript. This project features a clean UI, persistent storage using LocalStorage, and a progress-tracking celebration message.

## ✨ Features

* **Add Tasks:** Quickly add new items to your list.
* **Mark as Done:** Click on a task to toggle the "completed" state with a strikethrough and checkmark.
* **Remove Tasks:** Delete tasks individually using the '×' button.
* **Persistence:** Uses `LocalStorage` to save your tasks so they remain even after refreshing the page.
* **Celebration UI:** Displays a "Hurray! You've completed all your Tasks 🎉" message when all items are checked.
* **Responsive Design:** Optimized for both desktop and mobile viewports.

## 🛠️ Technologies Used

* **HTML5:** Structure and Semantic markup.
* **CSS3:** Custom styling, Flexbox for layout, and Linear Gradients.
* **JavaScript (ES6):** DOM manipulation and LocalStorage API.
* **Google Fonts:** 'Poppins' for a modern typography feel.
